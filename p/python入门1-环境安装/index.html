python包下载并安装

步骤一：访问python官网，下载安装包

步骤二：此处我们寻找python3.8.5安装包下载（推荐安装最新版的python，截至本文档的编写时，python最新的版本为3.12.6）

步骤三：双击已下载的安装包，按照下面步骤进行安装

步骤四：打开命令提示符，输入python，若出现下面信息，则代表安装成功

Vs Code代码编写工具安装

此处不介绍pyCharm工具的安装和使用，初学者使用该工具过于臃肿，建议熟悉python的开发后，再学习pycharm工具使用方式

步骤一：访问Vs Code官网，下载软件安装包

点击后，会自动下载最新版的vs code工具。

步骤二：双击安装包，安装vs code

vscode 插件安装(python)

步骤三：安装适合开发python的vs code插件。

① 安装简体中文插件，在插件商店输入chinese，按回车键搜索，并点击Install安装

安装完毕后，右下角出现下面界面，点击按钮重启vs code编辑器

vs code重启后，界面语言变成简体中文。

② 安装适合python开发的插件，在插件搜索栏输入python

安装完毕后，出现下面界面代表安装成功
